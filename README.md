# Algorithms-on-graphs

В этом репозитории хранится техническая реализация алгоритмов для доклада по дискретной математике "Алгоритмы на графах при оптимальной трассировке водопроводных путей"

Представлена реализация алгоритмов:
-Дейкстры
-Краскала
-A*
-Прима

Все алгоритмы реализованы на C++
